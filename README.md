# Node 18 LTS webapp

Minimal Node.js webapp targeting Node 18 LTS. Uses the built-in http module and includes a simple test.

Getting started

- Ensure you have Node 18 installed. .nvmrc included for nvm users.

Run server:

```bash
node src/server.js
# or
npm start
```

Run test (starts server on ephemeral port and checks GET /):

```bash
npm test
```
